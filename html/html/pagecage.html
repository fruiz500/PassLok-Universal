Please enter the URL in the box and click Load. Click Hide to hide this bar.

The page displayed is running in an isolated frame so no cookies are loaded, no history is recorded, and it is invisible to other extensions.

You type a new URL in the box, or select a PassLok web app or secure search engine from the drop-down list.

To load another page, refresh or click Clear. Links on page may load on a separate tab, but you can maintain security by right-clicking on them and selecting "Open Link in Cage."

This all works by loading the page into an iframe element in order to prevent other extensions from seeing anything typed into or produced by the page. Warning: this entails ignoring X-frame headers, which exposes the loaded page to clickjacking attacks.

Sometimes a website may refuse to load. In this case, you may want to install the "Ignore X-Frame headers" extension, from which this page has borrowed some code. This is the link for Chrome, and for Firefox.
